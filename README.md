# 遠傳加盟門市評論工具

## 功能
- 顯示 90 家遠傳加盟門市的 Google 評論數
- 按下「更新評論數」即時抓取最新評論數（Google Places API）
- 按下「下載報表」匯出 Excel 檔案

## 使用方式
1. 開啟網頁後，等待資料載入完成
2. 點擊「更新評論數」更新最新評論數
3. 點擊「下載報表」下載 Excel 檔案

## 部署方式
1. 將 `index.html` 和 `store_list_with_reviews.json` 上傳至 GitHub Repository 根目錄
2. 進入 Repository → Settings → Pages → Source → main → /(root)
3. 儲存後，GitHub Pages 網址會顯示在設定頁面
